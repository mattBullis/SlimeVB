void Robot::move ( int  ,
std::vector< Light >  ,
std::vector< Robot >   
)

The move function takes in vectors for both the lights and robots in the simulation, and the elapsed time in order to determine how far and in what direction the robot should move. It first gathers data from the light sensors and the robot sensors in order to determine what speed the left and right wheels should be. After this it checks to position to see if it has gone off screen, and if so it will wrap the robot around to the other side. The last portion gets the distance the robot should travel, rotates the robot, and then translates, completing the move call.

Parameters
intelapsedTime passed in to calculate translate distance
vector<Light>lights passed in to calculate speed in the sensors
vector<Robot>robots passed in to factor in robots into the speed
